DynaMate
Haiku GamePlay

Installing and Uninstalling
To install first unzip the archive. Then, cd into the "DynaMate" folder and run "make". You should now be able to run the command "DynaMate" to start the game.

To uninstall just remove the Dynamate folder (with contents).

Starting and Quiting

There are three ways to start playing.
Doubleclick a level
Drag a level onto the Dynamate application (while not running)
Drag a level onto the Dynamate Window
To quit Dynamate press cmd - Q
